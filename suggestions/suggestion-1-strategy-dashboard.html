Suggestion #1 — What Changed & Why

1. Brand-Aligned Color System

Replaced the current #0f172a/#1e293b with the deeper #0B0D14/#111322 from hylo.so — this matches the main app's near-black terminal aesthetic. The current site reads as "Tailwind dark mode" rather than "Hylo brand".

2. Typography: Outfit + JetBrains Mono

Every numeric value now uses Outfit 800 (letter-spacing: -0.5px) — they're the dominant visual on each card, not competing with labels. Labels and tags use JetBrains Mono 500 at 8–10px with aggressive uppercase + tracking, creating a clear data/label hierarchy that matches hylo.so's monospace label style.

3. Card Redesign — Tighter & More Scannable

Before: Cards used large padding (1.5rem), tall metric boxes, and 16px+ border-radius — felt spacious but wasted vertical space, especially at 26 cards.
After: Reduced to 18px body padding, 8px inner gaps, 14px card radius. Each card is ~15% shorter while keeping all data. The thin accent stripe at the top (1.5px gradient) replaces the thick 3px solid border-top for subtlety. Hover state adds a corner ambient glow (hylo.so style) instead of the current heavy box-shadow.

4. Points Footer — Amber Band

The points section is now a distinct amber-tinted footer bar at the card bottom rather than a box inside the card body. It creates a visual "callout" that immediately draws the eye to projected earnings — the metric most users care about.

5. Table View (New)

Added a toggleable table view grouped by source (Exponent / Rate-X), with expandable rows for detailed metrics. Power users monitoring 26 assets will prefer this — they can see leverage, yield, decay, risk, and points across all assets without scrolling through cards. Rows expand to show the full detail (recovery yield, last day YT value, lower range) that the card view shows inline.

6. Navbar — Contained Tab Strip

Replaced the floating individual buttons with a contained tab strip (background surface, shared border-radius 10px, 4px internal padding). Active tab uses solid violet with glow shadow — matching the convention on hylo.so's nav. Added a Live pill on the right with a pulsing teal dot.

This mockup preserves all existing data fields and metrics — nothing is removed. Only layout, spacing, typography, and color have been adjusted to better match hylo.so's brand identity.
