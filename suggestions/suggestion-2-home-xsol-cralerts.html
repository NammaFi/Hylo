Suggestion #2 — What Changed & Why

1. Live Ticker Bar (Home — NEW)

A scrolling ticker beneath the navbar showing real-time protocol stats (CR, leverage, TVL, SOL price, xSOL price) with % changes. This is common in DeFi dashboards and immediately signals "live data" without users clicking into any page. Uses JetBrains Mono for a terminal-data feel.

2. Home Hero — Brand-Split Title + Stats Pills

The title now splits "Hylo" (gradient) from "Community Toolkit" (white) on two lines for more visual weight. Below, three stats pills show pools tracked, data sources, and current CR — giving users instant context before clicking through. A subtle ✦ COMMUNITY-BUILT tag adds legitimacy.

3. Home Tool Cards — Color-Coded Accent Stripes + Badges

Each card gets a distinct accent stripe (violet / teal / amber / slate) matching the design system. Top-right badges (Live Data, Real-time, New) tell users the card's status at a glance. The Strategy card includes a mini bar-chart preview strip to hint at the dashboard's density.

4. Home — Protocol Health Snapshot (NEW)

A live mini-dashboard card next to CR Alerts showing current CR (big green number), leverage, TVL, xSOL price, and stability pool. Users get a protocol health check without leaving the home page. This pulls live data from the same xSOL metrics API.

5. xSOL Metrics — Sub-Nav Tabs + Daily Change Badges

Added xSOL Metrics / CR Alerts sub-nav tabs at the top, since both pages share protocol data. The three hero metric cards now show daily change badges (↑ +0.1× today) so users can see movement at a glance. Edit icons use the design system's surface-3 color instead of the current raw opacity approach.

6. xSOL Metrics — CR Progress Bar + TVL Chart Hint

The Collateral Ratio card gets a color-graded progress bar (red → amber → green → teal) with labeled zones. This instantly communicates protocol health without requiring users to mentally map the number. The TVL card has a mini bar-chart placeholder to hint at trend direction.

7. xSOL Break-Even — Cleaner Card Layout

The calculator now has a section divider label ("Break-Even Calculator") separating it from metrics above. Input uses a proper prefix slot ($) with focus ring. Result uses a gradient number treatment matching hylo.so's brand. An "Auto-updating" badge signals live calculation.

8. CR Alerts — Status Banner + Visual Gauge

A persistent CR status banner at page top shows current ratio + a mini color-graded gauge with "Danger / Warning / Safe / Healthy" zones. Users immediately know the protocol state before configuring alerts. This replaces the current plain text header.

9. CR Alerts — Quick Preset Chips + Better Threshold UX

Added color-coded preset chips (120% Danger, 130% Warning, 135% Caution, 140% Watch, 150% Comfortable) so users can one-click common thresholds instead of only typing. Saved thresholds show as styled cards with severity dots and descriptions, making the risk tier immediately visible.

10. CR Alerts — Numbered Steps + Bottom Status Bar

The two-step flow (1. Connect, 2. Thresholds) now uses numbered circles and proper card containers with metadata labels ("Required first", "Add multiple thresholds"). A persistent bottom bar shows connection status and CTA, so the connect button is always accessible even after scrolling past Step 1.

All existing data fields and functionality are preserved — nothing removed. Only layout, visual hierarchy, and UX flow have been improved to match hylo.so's brand identity.
