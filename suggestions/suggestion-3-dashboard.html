Suggestion 3 — Dashboard Card Refinements

  1. Removed Gradient top stripe on cards — the .ac::before gradient line is gone. Cards now have a clean, flat top edge.
  2. Changed No more boxes-within-boxes — inner metric boxes (.pm-item, .sm-box, .ag-item) now use transparent background (same as card) with a subtle 1px solid var(--border-2) outline border instead of background: var(--bg-base) (a darker fill).
  3. Removed Colored left borders on primary metric boxes (green/blue/violet left stripes) — unnecessary visual noise.
  4. Changed Simplified color system — reduced from ~7 colors to 3: green = positive values, red = negative/risk, white = primary metrics. Price Range changed from purple to neutral muted. Cleaner, less competing for attention.
